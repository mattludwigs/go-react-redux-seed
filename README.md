# go-react-redux-seed

Seed files to a Go, React, and Redux project

## Deps

1. Node 4+
2. Go 1.5+
3. glide

## Set up

```
npm install && glide up
```
or

```
npm run setup
```

## Run

two terminal tabs

```
npm run dev:client
```

```
npm run dev:server
```

## Build Client

```
npm run build:client
```
